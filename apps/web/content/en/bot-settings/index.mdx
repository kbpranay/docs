# Bot Settings

Bot Settings is where you configure the fundamentals of how a bot behaves — its name, API access, language handling, and how it classifies user messages.

Open it from the **Bot Settings** icon in the side menu bar.

## Sections in Bot Settings

The Bot Settings page has three main tabs:

| Tab | What you can do |
|---|---|
| **Bot Info** | Rename your bot, copy the API key, delete the bot (Team Owner only) |
| **DocBrain configs** | Configure RAG settings for DocBrain bots (see [DocBrain](/intents-entities/docbrain)) |
| **Classifier** | Enable or disable ML classification and Elasticsearch |

The Multilingual configuration also lives within Bot Settings for Admins, Team Owners, and Managers.
