# Configuración del bot

Bot Settings es donde configuras los fundamentos del comportamiento de un bot — su nombre, acceso a API, manejo de idiomas y cómo clasifica los mensajes del usuario.

Ábrelo desde el icono **Bot Settings** en la barra lateral del menú.

## Secciones de Bot Settings

La página Bot Settings tiene tres pestañas principales:

| Pestaña | Qué puedes hacer |
|---|---|
| **Bot Info** | Renombrar tu bot, copiar la API key, eliminar el bot (solo Team Owner) |
| **DocBrain configs** | Configurar los ajustes RAG para bots DocBrain (ver [DocBrain](/intents-entities/docbrain)) |
| **Classifier** | Habilitar o deshabilitar la clasificación ML y Elasticsearch |

La configuración Multilingual también vive dentro de Bot Settings para Admins, Team Owners y Managers.
