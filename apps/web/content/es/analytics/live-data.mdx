# Datos en vivo

Live Data muestra las conversaciones a medida que ocurren. Es la forma más rápida de detectar problemas tras un cambio recién desplegado, o de monitorizar un lanzamiento destacado en tiempo real.

## Qué incluye

Live Data muestra cada conversación activa actualmente en tus integraciones — chatbot web, llamadas de voz, WhatsApp y cualquier canal conectado.

Para cada sesión puedes ver:

- Origen y canal
- Hora de inicio
- Mensaje actual del usuario
- Respuesta actual del bot
- Si el usuario ha solicitado un agente

## Acceso por rol

La página Live Data es accesible para:

- Team Owner
- Manager
- AIDA Admin
- AIDA User
- AIDA Dashboard

El rol AIDA Dashboard tiene acceso de solo lectura, útil para operaciones de soporte o equipos de customer success que necesitan visibilidad sin derechos de edición.

## Usar Live Data durante despliegues

Un buen patrón tras lanzar un cambio:

1. Abre Live Data en una pestaña del navegador.
2. Filtra por el canal donde está activo tu cambio (por ejemplo, Webpage).
3. Busca patrones inesperados — picos repentinos de fallback, conversaciones más largas de lo habitual, más transferencias a agente.
4. Si algo parece ir mal, consulta la [página de entrenamiento](/intents-entities/train-conversations) para las sesiones específicas.

Para análisis históricos o agregados, usa [Informes e insights](/analytics/reports-insights) en su lugar.
