# Integrations (Flows)

**Flows** is AIDA's visual automation builder. It lets you drag-and-drop interconnected nodes to orchestrate everything from an API call to a full multi-step AI agent — without leaving AIDA.

Use Flows to:

- **Call external APIs** (CRM systems, ticketing, payments, translation services)
- **Query databases** with SQL, CSV tables, or knowledge bases
- **Plug in AI agents** — LLMs, RAG, classifiers, entity extraction, fully autonomous agents with tools
- **Build multi-step logic** — branching, templating, delays, retries, fallback handling
- **Hand off conversations** to a live agent when the bot can't resolve

## Why use Flows?

- **Visual, not code-heavy** — design complex logic by wiring nodes instead of writing code
- **Everything in one place** — no more switching tabs between your bot builder and integration tooling
- **Versioned & auditable** — every change is logged with who did it and when; revert any time
- **Lockable** — lock a flow to prevent accidental edits when it's running in production
- **Preview at design time** — card nodes show inline previews so you can validate layouts before shipping

## How Flows fit with the rest of AIDA

- **Intents & Entities** understand what a user *wants* → route to a flow
- **Flows** execute the integration logic → call systems, fetch data, generate responses
- **Channels** deliver the final response to the user (web widget, voice, WhatsApp)

From an intent, you can launch a flow as a response action: open the intent's **Response → Action** panel, pick **Flow** from the dropdown, and either create a new flow or paste an existing route ID.

<Callout type="info">
  **Channels vs. Integrations** — [Channels](/channels) are where your bot *talks* to users (Web, Voice, WhatsApp). Integrations (this section) are how your bot *connects to external systems* via Flows.
</Callout>

## Accessing Flows

Two routes from AIDA:

1. **Bot Selection page** → click **Flows**
2. **Mega Menu** (top navigation bar) → **Flows**

You can also access a specific flow by pasting its route ID into an Intent → Response → Action.
