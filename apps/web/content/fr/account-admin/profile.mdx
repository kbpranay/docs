# Profil et mot de passe

Les paramètres de votre profil se trouvent dans **Profile settings → Account** depuis le menu déroulant en haut à droite.

## Mettre à jour votre nom

<Steps>

### Ouvrez Profile Settings

Cliquez sur votre icône de profil dans le coin supérieur droit.

### Allez sur Account

Cliquez sur l'onglet **Account**.

### Modifier

Cliquez sur **[Edit]** à côté du champ My Profile.

### Mettez à jour et enregistrez

Modifiez votre prénom ou votre nom, puis cliquez sur **[Save]**.

</Steps>

<Callout>
  Votre adresse e-mail est l'identifiant de votre compte et **ne peut pas être modifiée**. Depuis cette page, vous pouvez également consulter — mais pas modifier — les rôles qui vous sont attribués.
</Callout>

## Changer votre mot de passe

<Steps>

### Ouvrez Profile Settings

Cliquez sur votre icône de profil → **Account** → **[Edit]**.

### Saisissez vos mots de passe

Remplissez les trois champs :

- **Current password** — votre mot de passe actuel
- **New password** — le mot de passe que vous souhaitez définir
- **Confirm password** — ressaisissez le nouveau mot de passe

### Enregistrer

Cliquez sur **[Save]**. Vous recevrez un e-mail de confirmation.

</Steps>

<Callout type="warning">
  Une fois votre mot de passe modifié, AIDA **vous déconnecte de la session en cours**. Reconnectez-vous avec votre nouveau mot de passe.

  Le bouton **Save** reste désactivé tant que les trois champs ne sont pas remplis.
</Callout>

## Fuseau horaire

Par défaut, AIDA utilise votre fuseau horaire préféré pour les horodatages dans les logs, l'analytique et les fonctionnalités planifiées. Vous pouvez le modifier depuis la page Account.

<Steps>

### Ouvrez Profile Settings → Account → Edit

Même point de départ que pour le changement de mot de passe.

### Sélectionnez un fuseau horaire

Cliquez sur le menu déroulant **Timezone** et choisissez l'option qui vous convient.

### Enregistrer

Cliquez sur **[Save]**.

</Steps>

<Callout>
  La déconnexion réinitialise votre fuseau horaire à **Asia/Calcutta**. Vous devrez le redéfinir lors de la prochaine connexion.
</Callout>
